## Calculadora

### Breve Descripción
- Calculadora estándar funcional con doble motivo de interfaz (Tema claro/Tema oscuro)
- Proyecto hecho enteramente en Java SE 8
  - Java with Ant
  - JFrame
  - Motor de JavaScript Nashorn
- IDE Apache NetBeans 12
